
# Rent A Car

Rent A Car is a backend API project developed using Spring Boot.

This project is designed to demonstrate core backend development concepts such as layered architecture, RESTful API design, and service-repository structure. In the initial phase, data is managed using an in-memory approach, allowing focus on application structure before integrating a database.

## Technologies Used
- Java 17
- Spring Boot
- Spring Web MVC
- Spring Validation
- Lombok
- OpenAPI / Swagger UI
- Maven

## Project Purpose
The main goal of this project is to learn and apply backend development principles by building a car rental system API. It provides a scalable structure that can be extended with database integration and additional features in the future.

## Features
- Layered architecture (Controller, Service, Data Access)
- RESTful API endpoints
- In-memory data handling (no database required for now)
- API documentation and testing with Swagger UI
- Ready for future database integration

## Future Improvements
- Database integration (PostgreSQL)
- CRUD operations for cars and rentals
- Authentication & authorization (JWT)
- Advanced filtering and search features
